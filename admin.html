운영 순서 요약

① 고객 문자 수신 → ② [주문 장부]에 붙여넣어 등록 → ③ 통장에서 주문번호·금액 확인 후 상태를 '입금 확인'으로 → ④ [발송 문자 복사]로 열람 링크 보내고 '발송 완료'로. 페이지 내용 수정은 [페이지 편집]에서, 수정 후엔 반드시 다운로드 → GitHub 업로드!

📤 업로드 탭에서 페이지를 고르고, 글자를 클릭해 수정하세요. 수정 후 [다운로드] → [업로드]에 드래그하면 1분 내 반영!

주문 목록

배지를 누르면 상태가 바뀝니다: 입금 확인 대기 → 입금 확인 → 발송 완료. ⚠️ 이 장부는 지금 이 브라우저에만 저장되니, 주기적으로 [도구]에서 백업하세요.

열람 링크 발송 문자 만들기

전자책 열람 링크를 한 번 저장해 두면, 주문 장부의 [발송 문자 복사] 버튼이 이 링크로 문자를 만들어줍니다.

장부 백업 · 복원

장부는 이 브라우저에만 저장됩니다. 컴퓨터를 바꾸거나 브라우저 데이터를 지우면 사라지니 주기적으로 백업 파일을 받아두세요.
